Descrizione dettagliata

Interfaccia base per tutti gli elementi riproducibili del sistema musicale.

Definisce il contratto comune per entità singole (come le tracce) e per entità aggregate (come le playlist). Grazie a questa interfaccia, i controller della UI e il motore del Media Player possono trattare qualsiasi sorgente in modo polimorfico e trasparente, senza dover implementare logiche separate per brani o playlist.

◆ getDisplayName()

Map< String, String > com.example.gruppo04.interfaces.PlayableSource.getDisplayName ( )

Recupera il dizionario dei metadati descrittivi della sorgente.

Restituisce le informazioni necessarie per popolare dinamicamente i componenti dell'interfaccia grafica (etichette, tabelle, player). L'implementazione deve garantire chiavi coerenti (es. "Titolo", "Autore" per le tracce; "Nome", "Brani Totali" per le playlist).

  • Restituisce
    Una Map<String, String> contenente le coppie chiave-valore da mostrare nella UI.

Implementato in com.example.gruppo04.model.PlaylistImpl, e com.example.gruppo04.model.TrackImpl.

◆ getTracks()

List< Track > com.example.gruppo04.interfaces.PlayableSource.getTracks ( )

Recupera la sequenza di brani associata a questa sorgente.

Il comportamento varia in base all'implementazione:

  • Se la sorgente è una singola Track, restituisce una lista di un solo elemento (se stessa).
  • Se la sorgente è una Playlist, restituisce l'elenco completo dei brani contenuti.
  • Restituisce
    Una List<Track> rappresentante la coda da riprodurre.

Implementato in com.example.gruppo04.interfaces.Playlist, com.example.gruppo04.model.PlaylistImpl, e com.example.gruppo04.model.TrackImpl.
