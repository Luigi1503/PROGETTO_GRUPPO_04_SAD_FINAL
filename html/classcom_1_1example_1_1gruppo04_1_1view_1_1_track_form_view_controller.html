◆ clearFormFields()

void com.example.gruppo04.view.TrackFormViewController.clearFormFields ( )
private

Ripulisce tutti i campi del form e ripristina lo stato iniziale.

Svuota i campi di testo, resetta la selezione della ComboBox, imposta valori di default per gli Spinner e blocca nuovamente l'interfaccia in attesa di un nuovo inserimento o selezione.

◆ estraiDatiDaMp3()

void com.example.gruppo04.view.TrackFormViewController.estraiDatiDaMp3 ( File file)
private

Legge e inserisce automaticamente i metadati estratti da un file MP3 nei campi del form.

Sfrutta la libreria mp3agic per analizzare i tag ID3v2 del file audio. Estrae informazioni come durata, titolo, artista, genere e anno di pubblicazione, popolando i relativi campi dell'interfaccia grafica.

  • Parametri
    fileL'oggetto File corrispondente al brano MP3 selezionato dall'utente.

◆ initialize()

void com.example.gruppo04.view.TrackFormViewController.initialize ( )

Inizializza i componenti grafici della View subito dopo il caricamento del file FXML.

Popola la ComboBox dei generi musicali, imposta i limiti per gli Spinner dell'anno (1900-corrente) e della durata (in secondi), applicando a quest'ultimo un converter personalizzato per mostrare il formato "MM:SS" invece dei secondi grezzi. Infine, disabilita i campi in attesa che l'utente importi un file o abiliti l'inserimento manuale.

◆ populateFormForEdit()

void com.example.gruppo04.view.TrackFormViewController.populateFormForEdit ( Track track)

Popola il form con i dati di una traccia esistente per consentirne la modifica.

Disabilita il bottone di "Aggiunta" e abilita quello di "Modifica". Sblocca i campi di input permettendo all'utente di alterare i valori correnti.

  • Parametri
    trackL'oggetto Track selezionato dall'utente nella tabella principale.
